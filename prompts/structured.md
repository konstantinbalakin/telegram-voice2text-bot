# Prompt: Structured Text

Твоя задача: структурировать текст голосовой транскрипции.

Исходный текст (сырая транскрипция):
{text}

## Требования

1. Исправить грамматические ошибки и опечатки
2. Добавить правильную пунктуацию (точки, запятые, вопросительные/восклицательные знаки)
3. Разбить на абзацы по смыслу (каждая новая мысль - новый абзац)
4. Выделить списки HTML-тегами если уместно (используй <ul><li>пункт</li></ul> для маркированных списков)
5. Сохранить весь смысл и все детали из оригинала
6. НЕ добавлять ничего от себя, только исправления
7. НЕ сокращать текст (это не резюме, а структурирование)
8. Сохранить стиль речи (неформальный/формальный)

## Форматирование HTML

Используй HTML-теги для форматирования текста:
- Жирный текст для важных моментов: <b>важный текст</b>
- Курсив для примечаний или акцентов: <i>примечание</i>
- Маркированные списки: <ul><li>пункт 1</li><li>пункт 2</li></ul>
- Нумерованные списки: <ol><li>пункт 1</li><li>пункт 2</li></ol>
- Код или технические термины: <code>термин</code>

Верни ТОЛЬКО исправленный текст с HTML-форматированием, без пояснений и комментариев.
