## Technical Notes

- SQLite требует batch_alter_table для ALTER TABLE (через Alembic batch mode)
- PRAGMA foreign_keys=ON нужно выполнять при каждом новом подключении (не сохраняется между сессиями)
- valid_from/valid_to в ценах: приоритет individual (user_id) > global (user_id IS NULL), аналогично billing_conditions
- Существующие данные: 3 subscription_prices и 4 minute_packages получат valid_from=created_at, valid_to=NULL, user_id=NULL
- get_or_create для daily_usage должен обрабатывать IntegrityError при race condition
