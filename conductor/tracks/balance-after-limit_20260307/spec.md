# Specification: Экран баланса при нехватке минут

**Track ID:** balance-after-limit_20260307
**Type:** Feature
**Created:** 2026-03-07
**Status:** Draft

## Summary

При недостатке минут или достижении дневного лимита — показывать короткое предупреждение, а следом автоматически отправлять экран /balance с кнопками покупки подписки и пакетов.

## Context

Сейчас при нехватке минут пользователь видит текстовое сообщение с инструкцией "Используйте /buy ... или /subscribe ...". Это требует ручного ввода команд. Нужно заменить на автоматическую отправку экрана баланса с inline-кнопками.

## User Story

Как пользователь, я хочу видеть экран баланса с кнопками покупки при нехватке минут, чтобы быстро перейти к пополнению без необходимости вводить команды вручную.

## Acceptance Criteria

- [ ] При "Недостаточно минут" (handlers.py) — короткое предупреждение + отдельное сообщение balance с кнопками
- [ ] При "Дневной лимит исчерпан" (orchestrator) — короткое предупреждение + отдельное сообщение balance с кнопками
- [ ] При "Дневной лимит почти исчерпан" (orchestrator) — короткое предупреждение + отдельное сообщение balance с кнопками
- [ ] Экран balance содержит кнопки "Купить подписку" и "Купить пакеты минут"
- [ ] Существующие тесты обновлены

## Dependencies

- BillingCommands._build_balance_text_and_markup() — используется для построения экрана баланса

## Out of Scope

- Изменения billing_service
- Новые UI-экраны
