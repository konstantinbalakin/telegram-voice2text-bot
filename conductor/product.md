# Product Definition

## Project Name

telegram-voice2text-bot

## Description

Telegram-бот для транскрипции голосовых сообщений с AI-обработкой текста.

## Problem Statement

Пользователи Telegram сталкиваются с двумя основными проблемами:

1. **Неудобно прослушивать голосовые сообщения** — не всегда есть возможность включить звук (совещания, транспорт, шумные места)
2. **Трудно извлечь смысл из длинных голосовых** — длинные аудиосообщения сложно переслушивать, искать ключевую информацию, структурировать

## Target Users

Пользователи Telegram, которые часто получают голосовые сообщения и нуждаются в быстром текстовом представлении с возможностью интеллектуальной обработки.

## Key Goals

1. **Быстрая и качественная транскрипция** — точное распознавание речи через OpenAI Whisper API
2. **Умная обработка текста** — структурирование, резюмирование, форматирование через DeepSeek V3
3. **Поддержка больших файлов** — обработка аудио до 2 ГБ через Telethon
