# Voice2Text Bot - Web Version

## Проект
Сервис для автоматической транскрибации голосовых сообщений в текст с AI-обработкой результата.

## Основные фичи

### Загрузка и транскрибация
1. **Загрузка аудио** — голосовые сообщения, файлы до 2 GB (MP3, M4A, WAV, OGG, OPUS)
2. **Мгновенная транскрибация** — автоматическое распознавание речи (OpenAI Whisper)
3. **Интерактивные кнопки** — 3 действия над текстом:
   - 📋 **Структурировать** — разбить на логические блоки, заголовки, списки
   - ✨ **Сделать красиво** — улучшить текст (грамматика, стиль)
   - 💡 **О чем этот текст** — сгенерировать резюме и ключевые тезисы

### Экраны
- **Главная**: форма загрузки + список последних транскрибаций
- **Результат**: исходный текст + 3 кнопки действий + редактирование
- **История**: поиск по транскрибациям, экспорт в PDF/DOCX

## Система биллинга и подписок

### Бесплатный уровень
- 10 минут в день
- 60 бонусных минут при регистрации

### Подписки
- **Pro (неделя)** — 80 ₽
- **Pro (месяц)** — 180 ₽
- **Pro (год)** — 1500 ₽
- Преимущество: 30 минут в день вместо 10

### Пакеты минут
- **50 минут** — 100 ₽
- **100 минут** — 175 ₽
- **500 минут** — 750 ₽
- **5000 минут** — 5000 ₽

### Способы оплаты
- Карта РФ / СБП (через YooKassa, чеки 54-ФЗ)

### Управление балансом
- **Экран баланса**: отображение дневного лимита, бонусных минут, пакетных минут, срока годности
- **Экран покупки**: каталог подписок и пакетов минут
- **История платежей**: список покупок и подписок

### Персонализация
- Персональные цены для конкретных пользователей
- Персональные дневные лимиты
- Срок годности минут с напоминаниями

## Дополнительные фичи
- Автосохранение результатов
- Копирование в clipboard
- Редактирование текста после обработки
- Прогресс-индикатор при загрузке больших файлов
- Умные уведомления о лимите (с учетом дополнительных минут)
